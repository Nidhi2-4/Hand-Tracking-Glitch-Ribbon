# Hand-Tracking-Glitch-Ribbon

An interactive, real-time creative coding project that maps human hand movements to dynamic, deforming digital textures with a futuristic cyber glitch aesthetic. Inspired by an Instagram reel and built to push the boundaries of computer vision in live visuals.

---

## Overview

This project integrates **MediaPipe's** machine learning capabilities with **TouchDesigner's** node based visual programming environment. By capturing live webcam input, the system tracks hand coordinates to dynamically stretch, warp, and deform visual ribbon textures live on camera. 


---

## Tech Stack

* **TouchDesigner**
* **MediaPipe** 
* **Python** 
* **GLSL / Shaders** 

---

## How to Run

1. Clone or download this repository.
2. Ensure you have **TouchDesigner** installed.
3. Open the `Real-time Hand Tracking Glitch Ribbon.toe` file.
4. Connect your webcam, ensure the MediaPipe tracking node is active, and start moving your hands!
